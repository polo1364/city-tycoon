# 🎲 大富翁遊戲 - 完整版

一個功能豐富、視覺精美的網頁版大富翁遊戲，支援多人對戰、智能 AI、多地圖選擇和華麗卡牌系統。

## 📸 遊戲截圖

遊戲採用全螢幕設計，棋盤佔據整個視窗，中央控制面板可收起，提供最佳遊戲體驗。

## ✨ 核心功能

### 🎮 遊戲系統

**新遊戲流程**
- 點擊「新遊戲」按鈕開始
- 選擇遊戲人數（2人、3人、4人對戰）
- 選擇你的角色（21個獨特角色）
- AI 自動隨機選擇剩餘角色
- 選擇地圖（5個地區可選）
- 開始遊戲！

**基本玩法**
- 擲骰子移動棋子
- 購買地產並收取租金
- 升級房屋增加收入
- 使用卡牌獲得優勢
- 讓對手破產獲勝

### 進階技巧

**1. 地產投資策略**
- 優先購買同色組地產，形成壟斷
- 橘色組（日月潭、阿里山、溪頭）性價比最高
- 車站收益穩定，建議早期購買
- 深藍組（澎湖、馬祖）價格高但回報率低

**2. 房屋升級策略**
- 先將同色組升級到2棟，再考慮更高等級
- 優先升級對手可能踩到的地產
- 保留足夠現金應對租金支付
- 旅館回報率最高，但投資巨大

**3. 卡牌使用技巧**
- 保護卡在關鍵時刻使用（避免入獄或高額租金）
- 傳送卡可用於避開對手的高級地產
- 雙倍收租卡在對手即將踩到你的旅館時使用
- 交換位置卡可讓對手踩到你的陷阱

**4. AI 對抗策略**
- 激進型 AI：阻止其完成壟斷
- 保守型 AI：積極擴張，搶佔先機
- 平衡型 AI：穩健對抗，避免冒險
- 投機型 AI：利用其現金流不足的弱點
